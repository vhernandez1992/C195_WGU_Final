Application Directions:
    Start by logging in using a valid username and password from the company provided database.
    At the main screen, choose between going to the customers or appointments screen, or logging back out to the login screen.
    In the customers screen, you can view the entire list of existing customers, add/delete customer or update an existing customer.
    In the appointments screen, you can view the list of appointments, add/delete an appointment or update and existing appointment.
    The reports tab in the appointments screen lets you run a report on different filters.

Additional Report:
    A 4th report chosen was location, which would allow for a report to be ran based on where the appointment would be taking place.
